# Voice-Activated Navigation Assistant for the Visually Impaired

An AI-powered, voice-activated mobile application designed to assist visually impaired individuals in navigating their surroundings safely by detecting real-time objects and hazards.

---

## 📝 Project Description

Navigating everyday environments poses significant safety and independence challenges for visually impaired individuals. This project introduces a hands-free, voice-activated mobile assistant that acts as an extra pair of "eyes." 

By combining mobile convenience with cutting-edge artificial intelligence, the application processes audio commands and environment visuals to provide immediate feedback to the user.

### Key Features
* **Voice-Activated Commands:** Completely hands-free operation designed for accessibility.
* **Real-Time Hazard & Object Detection:** Identifies obstacles, terrain changes, and potential hazards in the user's path.
* **Spatial Reasoning Feedback:** Powered by generative AI, the app doesn't just name objects—it explains where they are positioned, helping the user make safe navigation decisions.

### Tech Stack
* **Frontend:** React Native (Expo) for a cross-platform mobile user interface.
* **Backend:** Python Flask server to handle routing, audio streams, and processing logic.
* **AI Engine:** Gemini 1.5 Pro API for complex visual analysis, spatial reasoning, and conversational feedback.
